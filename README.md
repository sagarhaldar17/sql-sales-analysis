# SQL Data Analysis Projects

## Project 1: Sales Data Analysis

### Project Overview
This project analyses sales data using SQL to extract business insights, including revenue trends, customer behaviour, and product performance.

### Tools Used
- SQL (MySQL)

### Key Analysis
- Total revenue calculation
- Top-performing products
- Customer spending analysis
- Category-wise revenue
- Monthly sales trends

### Key Insights
- Total revenue generated is ₹147,000.
- Electronics category contributes ₹128,000 (~87%) of total revenue, making it the top-performing category.
- Top 2 customers (Amit and Rahul) contribute ₹95,000 (~65%) of total revenue.
- Monthly analysis shows revenue decreased from ₹102,000 in January to ₹45,000 in February.

### Skills Demonstrated
- SQL Queries (SELECT, WHERE, GROUP BY, ORDER BY)
- Data Analysis
- Data Aggregation

## Project 2: Customer Orders Analysis (JOIN)

### Project Overview
This project analyses customer and order data using SQL JOIN operations to understand customer spending patterns and city-wise sales performance.

### Key Analysis
- Customer order details using JOIN
- Total spending per customer
- Identification of top customers
- City-wise sales analysis
- Monthly revenue trends

### Key Insights
- Total revenue generated is ₹35,000.
- Rahul is the top customer with a total spending of ₹13,000.
- Delhi generates the highest revenue at ₹17,000, followed by Mumbai and Kolkata.
- Monthly revenue shows a slight variation, with February generating the highest revenue (₹13,000).

### Skills Demonstrated
- SQL Queries (SELECT, WHERE, GROUP BY, ORDER BY)
- SQL JOIN (INNER JOIN)
- Data Analysis
- Data Aggregation
- Business Insight Generation
